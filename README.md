Progetto effettuato per sostenere l'esame di programmazione 3 all'università degli studi Parthenope.
Il progetto vuole simulare attraverso l'uso di design pattern, il trasferimento di vari tipi di dati(immagini, audio, testi), attraverso dei modem.
L'obbiettivo finale del progetto è calcolare il percorso minimo per raggiungere il modem di uscita attraverso l'algoritmo di djikstra.
L'applicazione attraverso Tomcat ha una piccola interfaccia grafica che simula il percorso dei dati.
